OS X下的特殊字符输入有很多快捷键，并且不用依赖第三方输入法，直接可用非常方便

* 美元，`shift + 4`
* 美分，`option + 4`
* 英镑，`option + 3`
* 人民币，`option + y`
* 欧元，`shift + option + 2`
* 波折号，`option + -` 或 `shift + option + -`
* 省略号，`option + ;`
* 约等于，`option + x`
* 度，`shift + option + 8`
* 除号，`option + /`
* 无穷大，`option + 5`
* 小于等于，`option + ,`
* 大于等于，`option + .`
* 不等于，`option + =`
* 圆周率Pi，`option + p`
* 正负，`shift + option + =`
* 平方根，`option + v`
* 总和，`option + w`
* 商标Trademark，`option + 2`
* 注册，`option + r`
* 版权，`option + g`
